  • Catégorie:Développement Web Backend
  • Travaux:Site Plastique en Moins
  • Détails:Site réalisé dans le cadre d'un exercice utilisant le JavaScript ou NodeJS. J'ai développé cette idée de carte collaborative liée à la diminution de l'utilisation de plastiques et je l'ai adapté en JavaScript. Les utilisateurs peuvent cliquer sur la carte et indiquer un lieu qui utilise peu de plastiques (ex: pas de sachets plastiques, pas de barquettes plastiques, etc.). Le backend correspond à la base de données stockant les informations des lieux indiqués par les visiteurs du site.
  • Technoligies utilisées:PHP, SQL, JavaScript, HTML 5, CSS 3, VS Code
  • Dates:octobre 2021- juillet 2022
